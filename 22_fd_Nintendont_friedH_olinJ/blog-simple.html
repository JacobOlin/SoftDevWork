Forms

The first thing we can look at are two basic types of form inputs: the radio input and the checkbox input.

We can create these inputs simply by creating a form, and giving the input attribute type = "radio" or type = "checkbox"

The other interesting thing to note is how it is easy to allign the two sets of buttons horizontally

To do this, you can simply make a div with attribute class="grid-x grid-padding-x", and this will make it so that
anything you put inside the div will be put next to each other horizontally where there is room.



Buttons

Buttons are really nice to format. It comes with a bunch of preset classes for buttons including "success button", "alert button", and "secondary button".

To have your buttons send you somewhere, it is the standard 'href=___'.
